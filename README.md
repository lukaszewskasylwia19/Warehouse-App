# Warehouse-App

Aplikacja Java Swing będąca interfejsem do bazy danych SQL Server.
Pozwala na wyświetlenie listy produktów, klientów oraz zamówień.
Dodatkow pozwala na dodawanie, usuwanie oraz edycję klientów i produktów.
